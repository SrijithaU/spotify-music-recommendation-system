# spotify-music-recommendation-system

This project builds a music recommendation system using Spotify song features.

## Features
- Content-based recommendation system
- Cosine similarity for song recommendations
- K-Means clustering for music grouping
- PCA visualization for cluster analysis

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Spotify song dataset with audio features including danceability, energy, tempo, and acousticness.

## How It Works
1. Load dataset
2. Select audio features
3. Normalize data
4. Compute cosine similarity
5. Recommend similar songs
6. Cluster songs using K-Means

## Visualization
Music clusters using PCA.
